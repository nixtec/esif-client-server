Server Utilities

Following server utilities are available:

  • delpass - delete college entry
  • dispass - disable college
  • enapass - enable college
  • inspass - insert college info
  • showpassdb - display college info
  • showregdb - dump a college database

Following helper scripts are available for ease of management:

  • rc.eduserver - control server

The following file contains various file and input formats:

  • format.html - input and file format
