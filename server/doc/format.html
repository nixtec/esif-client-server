      Start and End time for a college must be specified as follows:

      yyyy-mm-dd.HH:MM:SS

      For example,

      2005-11-15.18:31:01

Subject Code File:

      (This file contains subject code => name mapping information)
        subcode.txt


      File Format:
        code Name

      For example:

       101 Bangla 1st Paper

    
      
Special Subject Code File:

        special.txt


      File Format:
        code1
	code2
        ...

      For example:

      192

    
      
Pair Matching File:

      (This file contains subject code pair information)
        pair.txt


      File Format:
        code1 code2                  < both must be matched
	code2 code1                  < both must be matched
	code3 002 code7 code8        < all (excluding 002)  must be matched
	code3 001 code7 code8        < any two must be matched
        ...

      For example:
        101 102
        102 101
        192 002 191 193
        191 192
        193 192
        136 001 203 204
        ...

Server Host Information:

        website.txt

      File Format:
        host:ip

      'ip' is optional.
      Here I have placed the 'ip' field so that we can send form still if
      we don't have a DNS name registered. At first, the program will try
      to look up IP address of the 'host' specified in the file. If it fails
      to get the required IP address, it will use the ip specified in the
      file (if present) otherwise it will fail.

      The server host name must be specified in this file.

      For example:

      www.rajshahi-educationboard.org:202.92.52.125
