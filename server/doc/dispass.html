dispass - disable college.

    Usage: dispass [< passfile.txt]

      You can run this program only by typing its name. In that case you can
    interactively delete colleges as you type. Or, you can supply a filename
    as '< passfile.txt' during command execution. In that case, college
    codes will be read from that file. 'passfile.txt' is the file you specify
    to read college codes from. In both case, input text must be as following
    File Format standard.


    File Format:

    college_code1
    college_code2
    ...
    college_codeN
